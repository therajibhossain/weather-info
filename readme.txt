Weather Info displays your location weather with forcast.

== Description ==

Using Weather Info is so easy. It displays your current weather based on the location. Then set a widget Just make few settings and that's it.

Please note that, this plugin relies on openweathermap.org to get weather & forecast data and also relies on ipstack.com to get visitors` IP location.

weather info is showing the weather-data from api.openweathermap.org. please also refer to the terms of usage of openweathermap.org
Also it's using ipstack.com for getting the visitors location for displaying the weather

== Installation ==

Just install from your WordPress "Plugins > Add New" screen and all will be done automatically. Manual installation:

1. Upload the zip file and unzip it in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Settings > Weather Info` and setup your API Key from ipstack.com and APP ID from openweathermap.org.
4. Go to `Appearance > Widget > Weather Info` and enable the options that you need.
